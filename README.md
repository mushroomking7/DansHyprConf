My personal Hyprland config.

## v2.0 Changes:

- Added support for newer versions of hyprland (02/02/2026)

- Dynamic wallpaper swapping when changing workspaces

- Now able to change wallpapers with a hotkey

### Requirements: (May be subject to change)

- Hyprland

- Hyprpaper

### Installation guide:

1. Download v2.0 and extract the contents into the hyprland config folder. Typically found in '/home/yourusername/.config/hypr/'

2. Make the hyprpaper script run on startup. If you are using systemdthat would be 'systemctl enable hyprpaper.service'
